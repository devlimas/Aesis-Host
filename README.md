# Aesis-Host(Finalizado)

### Ideia do projeto:
Um lading page básica de hosting que oferece diversos planos

### O que este projeto faz?
 - Projeto prático para domínio de fundamentos do Tailwind CSS, HTML e CSS
 - Desenvolver o entendimento sobre flexbox e responsividade
 - Compreender a estruturação de uma página com HTML

### O que eu aprendi?
 - Utility-First: Como estilizar elementos sem criar arquivos .css gigantes.
 - Responsividade: Uso dos prefixos sm:, md: e lg: para adaptar a tela.
 - Entedimento melhor sobre flexbox e gridbox
